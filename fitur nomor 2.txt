
Fungsi menerima 1 argumen `nama`, misal: `"asep07"` ‚Üí akan dicek di semua platform.

---

### 2. **Menampilkan Header**

```python
print(Fore.LIGHTGREEN_EX + "[2] Pengecekan Username Sosial Media...")
```

Menampilkan info awal ke user dengan warna hijau (pakai modul `colorama`).

---

### 3. **List Platform**

```python
sosial = [
    "https://facebook.com",
    "https://twitter.com",
    ...
    "https://venmo.com"
]
```

Berisi **51+ URL platform** tempat akun user bisa terdaftar.
Format URL sudah diatur supaya cocok dengan struktur user masing-masing platform.

Contoh:

* `https://instagram.com/asep07`
* `https://github.com/asep07`
* `https://tiktok.com/@asep07`

---

### 4. **Looping Setiap Platform**

```python
for site in sosial:
    url = f"{site}/{nama}"
```

Script akan:

* Gabungkan URL + `nama` ‚Üí membentuk full URL akun
* Contoh: `https://github.com/asep07`

---

### 5. **Request ke URL**

```python
res = requests.get(url, timeout=6)
```

Melakukan **permintaan HTTP GET** ke URL tersebut, dengan timeout 6 detik.

---

### 6. **Cek Status Ketersediaan**

```python
if res.status_code == 200:
    print(Fore.GREEN + f"[AKTIF] {url}")
elif res.status_code == 404:
    print(Fore.RED + f"[TIDAK ADA] {url}")
else:
    print(Fore.YELLOW + f"[?] {url} [status: {res.status_code}]")
```

* **200 = username aktif** di platform itu
* **404 = tidak ditemukan (belum digunakan)**
* **Lainnya** bisa 403, 500, atau status aneh

---

### 7. **Tangani Error**

```python
except:
    print(Fore.RED + f"[ERROR] {url}")
```

Jika ada error jaringan atau domain, script tetap jalan dan hanya menampilkan `[ERROR]`.
